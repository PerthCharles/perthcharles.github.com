linux命令学习（十）：awk

awk命令

awk对比sed：awk主要针对行内数据进行操作，sed主要是以行为单位进行操作
命令语法格式： 

#awk '条件类型1{动作1} 条件类型2{动作2} ...' filename

awk过于复杂，所以以后慢慢练习。

1.awk 计算方差 

#awk '{a[++i]=$1;} END{for(i in a)sum += a[i];ave=sum/NR;for(i in a) delta += (a[i]-ave)*(a[i]-ave);print delta/NR}' tmp
